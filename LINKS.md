## Quick Commands to Get URLs

**Get EC2 IP:**
```powershell
cd terraform
terraform output ec2_public_ip
```

**Get Application URL:**
```powershell
cd terraform
terraform output app_url
```

**Get All Outputs:**
```powershell
cd terraform
terraform output
```
